# Public Research Snapshot

This directory is generated by `scripts/build_public_snapshot.py` from the
reviewed B-side research outputs.

The snapshot contains ten tokens and the numerical fields required by the
public read-only dashboard. It intentionally excludes pool addresses,
exchange lists, private notes, local history, and unsupported DEX direction
fields.

Regenerate it after research outputs change:

```bash
python3 scripts/build_public_snapshot.py
```

See `manifest.json` for row counts, content hashes, dates, and the source Git
commit. See `DATA_USAGE.md` before redistributing the snapshot.
